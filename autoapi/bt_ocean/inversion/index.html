class bt_ocean.inversion.ModifiedHelmholtzSolver(grid, alpha, beta=1)

Solver for the 2D modified Helmholtz equation,

\[( \beta ( \partial_{xx} + \partial_{yy} ) - \alpha ) u = b,\]

subject to homogeneous Dirichlet boundary conditions, using a finite difference discretization.
